# 认证 API

## POST `/api/auth/login`

密码登录。成功后设置 `access_token` 和 `refresh_token` HttpOnly Cookie。

请求：

```json
{
  "username": "admin",
  "password": "password123",
  "totp_code": "123456"
}
```

`totp_code` 仅在用户开启 TOTP 时必填。

响应 `200`：

```json
{
  "token_type": "bearer",
  "expires_in": 1800
}
```

常见错误：

| 状态码 | 说明 |
| --- | --- |
| 400 | 需要 TOTP 验证码 |
| 401 | 用户名、密码或 TOTP 错误 |

## POST `/api/auth/refresh`

刷新访问令牌。需要请求携带有效 `refresh_token` Cookie。

响应 `200`：

```json
{
  "token_type": "bearer",
  "expires_in": 1800
}
```

## POST `/api/auth/logout`

登出并清除认证 Cookie。

认证：需要登录。

响应：`204 No Content`

## POST `/api/auth/register`

创建新用户。

认证：需要当前用户为 superuser。

请求：

```json
{
  "username": "newuser",
  "email": "user@example.com",
  "password": "password123"
}
```

验证：

| 字段 | 规则 |
| --- | --- |
| `username` | 3-50 字符 |
| `email` | 合法邮箱 |
| `password` | 至少 8 字符 |

响应 `201`：

```json
{
  "id": 2,
  "username": "newuser",
  "email": "user@example.com",
  "is_active": true,
  "is_superuser": false,
  "totp_enabled": false,
  "created_at": "2026-01-01T00:00:00Z",
  "updated_at": "2026-01-01T00:00:00Z"
}
```

## GET `/api/auth/me`

获取当前用户信息。

认证：需要登录。

响应 `200`：

```json
{
  "id": 1,
  "username": "admin",
  "email": "admin@example.com",
  "is_active": true,
  "is_superuser": true,
  "totp_enabled": false,
  "created_at": "2026-01-01T00:00:00Z",
  "updated_at": "2026-01-01T00:00:00Z"
}
```

## POST `/api/auth/wallet/nonce`

获取钱包签名 nonce 和 message。

请求：

```json
{
  "wallet_address": "0x0000000000000000000000000000000000000000"
}
```

响应 `200`：

```json
{
  "nonce": "hex-nonce",
  "message": "Sign this message to login: hex-nonce"
}
```

## POST `/api/auth/wallet/login`

钱包签名登录，用户不存在时自动注册。

请求：

```json
{
  "wallet_address": "0x0000000000000000000000000000000000000000",
  "signature": "0x...",
  "message": "Sign this message to login: hex-nonce"
}
```

响应 `200`：

```json
{
  "user_id": 1,
  "wallet_address": "0x0000000000000000000000000000000000000000"
}
```

## POST `/api/auth/totp/setup`

生成 TOTP secret 和 Authenticator URI。

认证：需要登录。

响应 `200`：

```json
{
  "secret": "JBSWY3DPEHPK3PXP",
  "uri": "otpauth://totp/Drexor:admin?secret=...&issuer=Drexor"
}
```

## POST `/api/auth/totp/verify`

验证并启用 TOTP。

认证：需要登录。

请求：

```json
{
  "totp_code": "123456"
}
```

响应：当前用户信息。

## POST `/api/auth/totp/disable`

禁用 TOTP。

认证：需要登录。

响应：当前用户信息。

## POST `/api/auth/api-keys`

创建 API Key。明文 `secret` 只返回一次。

认证：需要登录。

请求：

```json
{
  "name": "local sdk",
  "expires_in": 2592000
}
```

响应 `201`：

```json
{
  "id": 1,
  "name": "local sdk",
  "key": "dk_xxx",
  "secret": "secret-only-returned-once",
  "expires_at": "2026-02-01T00:00:00Z",
  "created_at": "2026-01-01T00:00:00Z"
}
```

## GET `/api/auth/api-keys`

列出当前用户 API Key。

认证：需要登录。

响应 `200`：

```json
[
  {
    "id": 1,
    "name": "local sdk",
    "key": "dk_xxx",
    "is_active": true,
    "expires_at": null,
    "last_used_at": null,
    "created_at": "2026-01-01T00:00:00Z"
  }
]
```

## PUT `/api/auth/api-keys/{api_key_id}/revoke`

吊销 API Key。

认证：需要登录，且只能操作自己的 key。

响应：`204 No Content`

## DELETE `/api/auth/api-keys/{api_key_id}`

删除 API Key。

认证：需要登录，且只能操作自己的 key。

响应：`204 No Content`
