# 公共 API

## GET `/api/docs/health`

健康检查。

响应 `200`：

```json
{
  "status": "healthy"
}
```

## GET `/api/docs/info`

服务信息。

响应 `200`：

```json
{
  "name": "drexor-python",
  "version": "0.1.0",
  "description": "Drexor Python backend with authentication, database, Redis and WebSocket"
}
```

## 在线 OpenAPI

开发环境：
